# Dataset

This folder contains the combined and cleaned dataset for the supervised movie revenue prediction project.

## Combined And Cleaned Dataset

`comparison_outputs/Movies-Dataset.csv` is the project dataset created after combining the two raw movie sources, resolving overlapping movie records, and filtering the data for usable supervised learning rows with valid budget and revenue values.

This dataset is the base table used by the EDA, feature preparation, full workflow, and hyperparameter tuning notebooks.
